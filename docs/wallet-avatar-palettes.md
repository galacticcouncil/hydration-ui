# Wallet Avatar Palettes

Generated by `node scripts/extract-wallet-avatar-palettes.mjs`.

These palettes are used only for generated account marbles. Real wallet/provider logos stay unchanged.

The default account palette removes white because it created awkward bright areas in the marble avatars:

`#98AFFF`, `#53A4F3`, `#E53E76`, `#F9AFCA`, `#AAEEFC`

## Notes

- Substrate wallet providers use the Hydration account palette instead of logo-derived colors, including Polkadot.js, Talisman, SubWallet, Nova, Enkrypt, Manta, Fearless, PolkaGate, and Aleph Zero.
- EVM and non-Substrate ecosystem wallets keep logo-derived palettes where account marbles benefit from wallet-specific color, such as MetaMask orange, Rabby blue, Phantom purple, and Solflare yellow.
- WalletConnect keeps its own palette because it can be used for more than one account ecosystem.
